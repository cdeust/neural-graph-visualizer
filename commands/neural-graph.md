---
name: neural-graph
description: Launch the Neural Graph Visualizer
---

Use the `neural-graph-visualizer:open_visualization` tool to launch the interactive graph visualization in the browser.

If the user wants statistics, use `neural-graph-visualizer:get_stats` first to show a summary, then offer to open the visualization.

If the user wants to search, use `neural-graph-visualizer:search` with their query.
